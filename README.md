# C# Programming ExercisesThis repository contains a collection of C# console applications developed as part of my Unity and C# programming coursework. These exercises serve as a practical foundation for understanding core programming concepts, object-oriented design, and engine integration.Exercise OverviewThis collection covers a range of foundational topics:Basic I/O and Logic:Exercise 4: Calculates altitude changes based on user input.  Exercise 6: A "Hello, World!" implementation.  PrintAge: Demonstrates variable declaration and outputting an integer value.  ProgrammingAssignment1: Demonstrates outputting a list of chemical symbols to the console.  Data Types and Math:IntegerDataTypes: Calculates and displays minutes and seconds played using modulo and division operators.  Object-Oriented Programming (OOP) & Classes:Exercise 8: Demonstrates the use of custom classes (Die) and random number generation to simulate rolling dice and summing their values.  Exercise 9: Demonstrates the use of custom classes (Deck, Card) to shuffle, print, and manipulate a collection of cards.  UsingDeckClass: Demonstrates basic initialization and interaction with the Deck class.  How to RunClone this repository to your local machine.Open the desired exercise folder in your preferred C# IDE (e.g., Visual Studio).Ensure your environment is configured for .NET console applications.Build and run the Program.cs file associated with the specific exercise.AuthorEthane Masserey
